## 🔖 Sobre:
O projeto consiste em criar uma interface com uma To-Do list com a aplicação de drag-and-drop. Onde poderemos arrastar e soltar livremente os elementos entre as colunas.

---

## 🚀 Tecnologias usadas
- HTML;
- CSS;
- JavaScrip.
